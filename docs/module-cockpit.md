# Cockpit

Le Cockpit est la page d'accueil de la plateforme. Il fournit une vue d'ensemble de la posture de sécurité de votre organisation.

## Rôle du module

Le Cockpit centralise les indicateurs clés pour permettre un pilotage rapide et efficace. Il s'adresse principalement aux responsables sécurité et aux décideurs, mais reste accessible à tous les utilisateurs authentifiés.

## Ce que vous pouvez y faire

- **Consulter les indicateurs globaux** — Nombre d'objets, projets actifs, score moyen, tâches en attente
- **Suivre le Security Integrity Score** — Évolution du score global et par périmètre
- **Identifier les priorités** — Objets les moins conformes, tâches de remédiation urgentes
- **Accéder rapidement aux projets récents** — Liens directs vers les projets en cours

## Accès

Le Cockpit est visible par tous les utilisateurs authentifiés disposant d'au moins une permission de lecture sur les ressources affichées.
